Active Looking

A Pedagogical Approach to Multimodal Compositions

Multimodal Classroom

Based on a model proposed by both Iowa State University and Georgia Technical Institute featuring several modes Written Oral Verbal Electronic Nonverbal, creating the acronym WOVEN, a team at Georgia State University has been working on creating a new media literacy themed set of Composition courses. These courses rely on experimenting with processes, compositing and could benefit from ‘Active Looking‘ as a pedagogical practice to test out varying emerging practices.
